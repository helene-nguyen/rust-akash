## What's Changed

## akash v0.1.0 — Initial Release

First public release of akash, a cross-platform CLI tool for managing shell aliases.

### Highlights

- **One store, any shell** — Define aliases once in JSON, apply to PowerShell, Bash or Zsh
- **Auto-detection** — Detects your shell automatically, override with `--shell`
- **Safe config editing** — Block markers prevent overwriting your existing shell config
- **Interactive mode** — Menu-driven interface when run without arguments
- **Dual binary** — Use `akash` or `aka`
